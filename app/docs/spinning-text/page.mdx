# SpinningText

Easily animate text content or components in a circular motion with customizable properties such as width, degree of rotation, text style, and custom classes. The `SpinningText` component leverages `Framer Motion` for smooth animations.

## Examples

### Basic Usage

<ComponentCodePreview
  component={
    <SpinningText>
      <span>Your Text Here</span>
    </SpinningText>
  }
  filePath='core/spinning-text'
  hasReTrigger
/>

### Custom Width and Degree

<ComponentCodePreview
  component={
    <SpinningText width={200} degree={360}>
      <span>Custom Rotation -</span>
    </SpinningText>
  }
  filePath='core/spinning-text'
  hasReTrigger
/>

### Custom Text Styles and Classname

<ComponentCodePreview
  component={
    <SpinningText
      textStyle={{ fontSize: '24px', color: 'blue' }}
      className='custom-class'
    >
      <h1>Styled Text</h1>
    </SpinningText>
  }
  filePath='core/spinning-text'
  hasReTrigger
/>

### Use Components as Children

If you are using a component as a child, the component should pass any text via a `content` prop. This ensures the text can still be animated in a circular motion.

<ComponentCodePreview
  component={
    <SpinningText>
      <Title content={'GO HOME'} />
    </SpinningText>
  }
  filePath='core/spinning-text'
  hasReTrigger
/>

## Component API

### SpinningText

| Prop      | Type            | Default | Description                                                                  |
| :-------- | :-------------- | :------ | :--------------------------------------------------------------------------- |
| children  | `ReactElement`  |         | The text content or component to be animated in a circular motion.           |
| width     | `number`        | `200`   | Defines the width and height of the circular text container.                 |
| textStyle | `CSSProperties` | `{}`    | Custom styles for the text elements or components inside the circle.         |
| degree    | `number`        | `180`   | The degree of rotation to apply to the text around the circle.               |
| duration  | `number`        | `10`    | The duration of the full circular rotation animation.                        |
| className | `string`        | `''`    | A custom class name to apply styles to each rotating character or component. |
