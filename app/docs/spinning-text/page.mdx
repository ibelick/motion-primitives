# SpinningText

Easily animate text content in a circular motion with customizable properties such as width, degree of rotation, and text style. The `SpinningText` component leverages `Framer Motion` for smooth animations.

## Examples

### Basic Usage

<ComponentCodePreview
  component={
    <SpinningText>
      <span>Your Text Here</span>
    </SpinningText>
  }
  filePath='core/spinning-text'
  hasReTrigger
/>

### Custom Width and Degree

<ComponentCodePreview

## Component API

### SpinningText

| Prop      | Type            | Default | Description                                                    |
| :-------- | :-------------- | :------ | :------------------------------------------------------------- |
| children  | `ReactElement`  |         | The text content to be animated.                               |
| width     | `number`        | `200`   | Defines the width and height of the circular text container.   |
| textStyle | `CSSProperties` | `{}`    | Custom styles for the text elements inside the component.      |
| degree    | `number`        | `180`   | The degree of rotation to apply to the text around the circle. |
