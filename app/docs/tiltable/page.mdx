# Tiltable

A component that creates an interactive 3D tilt effect using spring animations. It responds to mouse movement to create a dynamic, depth-enhancing effect that makes UI elements more engaging. Built with Framer Motion, it provides smooth transitions and customizable rotation factors.

## Component API

### Border Trail

| Prop           | Type       | Default | Description                                          |
| :------------- | :--------- | :------ | :--------------------------------------------------- |
| className      | string     |         | Additional CSS classes for styling the border trail. |
| transition     | Transition |         | Custom transition settings for the animation.        |
| rotationFactor | number     | 15      | Controls the maximum rotation angle in degrees.      |
| isRevese       | boolean    | false   | Reverses the tilt effect's rotation direction.       |
