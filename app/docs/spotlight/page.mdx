# Spotlight

The Spotlight component creates a dynamic spotlight effect that follows the cursor, highlighting elements on the page. It's perfect for drawing attention to specific areas or creating an interactive experience.

## Component API

| Prop           | Type            | Default                           | Description                                        |
| :------------  | :-------------- | :-------------------------------- | :------------------------------------------------- |
| children       | React.ReactNode |                                   | The content to be wrapped by the spotlight effect. |
| className      | string          | ''                                | Additional CSS classes for styling the container.  |
| spotlightSize  | number          | 200                               | The size of the spotlight in pixels.               |
| color          | string          | 'rgba(255, 255, 255, 0.85)'       | The color of the spotlight effect.                 |
| opacity        | number          | 0.15                              | The opacity of the spotlight effect.               |
| blur           | number          | 40                                | The blur amount of the spotlight effect in pixels. |
| attachToParent | boolean         | false                             | If `true`, spotlight will only be visible when hovering over its parent component.   |
| springOptions  | SpringOptions   | `{ stiffness: 400, damping: 25 }` | Custom spring settings for the spotlight movement.  |
