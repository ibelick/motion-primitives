## Component API

### Spotlight

| Prop           | Type          | Default                        | Description                                    |
| :------------ | :------------ | :---------------------------- | :--------------------------------------------- |
| className     | string        | ''                           | Additional CSS classes for styling              |
| spotlightSize | number        | 200                          | Size of the spotlight effect in pixels          |
| color         | string        | 'rgba(255, 255, 255, 0.2)'   | Color of the spotlight gradient                |
| blur          | number        | 20                           | Blur amount in pixels                           |
| opacity       | number        | 1                            | Maximum opacity when hovered                    |
| springOptions | SpringOptions | `{ stiffness: 400, damping: 25 }` | Animation spring configuration            |
